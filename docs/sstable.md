# SSTable

## SSTable record

```
+---------------+---------------+---------------+-----------------+---------------+
| key size (2B) |      key      |   timestamp   | value size (2B) |     value     |
+---------------+---------------+---------------+-----------------+---------------+
```

## Data block

```
+--------------------------+-------------------------+----------------+
|      Records section     |     Offsets section     |  # of records  |
+--------------------------+-------------------------+----------------+
| Record |       ...       | Offset (4B) |    ...    |     # (4B)     |
+--------------------------+-------------------------+----------------+
```

## Index record

```
+---------------+-------------+-------------------+
| key size (2B) |   min key   | block offset (8B) |
+---------------+-------------+-------------------+
```

## SSTable file


```
+-----------------------------------------------------------------------+
|                            1. Data blocks                             |
+-----------------------------------------------------------------------+
| Data block | Data block | Data block |               ...              |
+-----------------------------------------------------------------------+
|                            2. Index block                             |
+-----------------------------------------------------------------------+
|      Records section     |      Offsets section     |  # of records   |
+-----------------------------------------------------------------------+
| Index record |    ...    | Offset (4B) |    ...     |     # (4B)      |
+-----------------------------------------------------------------------+
|                          3. Bloom filter                              |
+-----------------------------------------------------------------------+
|            4. Min key             |            5. Max key             |
+-----------------------------------------------------------------------+
| 6. Index block start offset (8B)  | 7. Bloom filter start offset (8B) |
+-----------------------------------------------------------------------+
|      8. Min key offset (8B)       |      9. Max key offset (8B)       |
+-----------------------------------------------------------------------+
|     10. Number of records (4B)    |      11. Data block size (4B)     |
+-----------------------------------------------------------------------+
```
