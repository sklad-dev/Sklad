# SSTable

## SSTable record

```
+---------------+- -----+-----------+-----------------+-------+-------+----------------+
| key size (2B) |  key  | timestamp | value size (2B) | flags | value | (optional TTL) |
+---------------+-- ----+-----------+-----------------+-------+-------+----------------+
```

### Record flags

| Bit | Name     | Description                                   |
|-----|----------|-----------------------------------------------|
| 0   | FLAG_TTL | If set, an 8-byte TTL value follows the value |
| 1-7 | Reserved | Reserved for future use                       |

### Tombstone

A tombstone (deleted record) is represented by `value size = 0`. In this case, the flags, value, and TTL fields are omitted:

```
+---------------+- -----+-----------+----------------+
| key size (2B) |  key  | timestamp | value size = 0 |
+---------------+-- ----+-----------+----------------+
```

## Data block

```
+--------------------------+-------------------------+----------------+
|      Records section     |     Offsets section     |  # of records  |
+--------------------------+-------------------------+----------------+
| Record |       ...       | Offset (4B) |    ...    |     # (4B)     |
+--------------------------+-------------------------+----------------+
```

## Index record

```
+---------------+-------------+-------------------+
| key size (2B) |   min key   | block offset (8B) |
+---------------+-------------+-------------------+
```

## SSTable file


```
+-----------------------------------------------------------------------+
|                            1. Data blocks                             |
+-----------------------------------------------------------------------+
| Data block | Data block | Data block |               ...              |
+-----------------------------------------------------------------------+
|                            2. Index block                             |
+-----------------------------------------------------------------------+
|      Records section     |      Offsets section     |  # of records   |
+-----------------------------------------------------------------------+
| Index record |    ...    | Offset (4B) |    ...     |     # (4B)      |
+-----------------------------------------------------------------------+
|                          3. Bloom filter                              |
+-----------------------------------------------------------------------+
|            4. Min key             |            5. Max key             |
+-----------------------------------------------------------------------+
| 6. Index block start offset (8B)  | 7. Bloom filter start offset (8B) |
+-----------------------------------------------------------------------+
|      8. Min key offset (8B)       |      9. Max key offset (8B)       |
+-----------------------------------------------------------------------+
|     10. Number of records (4B)    |      11. Data block size (4B)     |
+-----------------------------------------------------------------------+
```
