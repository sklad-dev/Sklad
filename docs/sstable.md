# SSTable

## SSTable record

```
+---------------+---------------+----------------------+---------------+
| key size (2B) |      key      | value size (2 bytes) |     value     |
+---------------+---------------+----------------------+---------------+
```

## Data block

```
+--------------------------+-------------------------+----------------+
|      Records section     |     Offsets section     |  # of records  |
+--------------------------+-------------------------+----------------+
| Record |       ...       | Offset (2B) |    ...    |     # (4B)     |
+--------------------------+-------------------------+----------------+
```

## Index record

```
+---------------+-------------+-------------------+
| key size (2B) |   min key   | block offset (4B) |
+---------------+-------------+-------------------+
```

## SSTable file


```
+-----------------------------------------------------------------------+
|                            1. Data blocks                             |
+-----------------------------------------------------------------------+
| Data block | Data block | Data block |               ...              |
+-----------------------------------------------------------------------+
|                            2. Index block                             |
+-----------------------------------------------------------------------+
|      Records section     |      Offsets section     |  # of records   |
+-----------------------------------------------------------------------+
| Index record |    ...    | Offset (2B) |    ...     |     # (4B)      |
+-----------------------------------------------------------------------+
|                          3. Bloom filter                              |
+-----------------------------------------------------------------------+
|            4. Min key             |            5. Max key             |
+-----------------------------------------------------------------------+
| 6. Index block start offset (4B)  | 7. Bloom filter start offset (4B) |
+-----------------------------------------------------------------------+
|      8. Min key offset (4B)       |      9. Max key offset (4B)       |
+-----------------------------------------------------------------------+
|                       10. Data block size (4B)                        |
+-----------------------------------------------------------------------+
```
